🎨 ByteyBuddy Color Toolkit

A modern, lightweight, and fully responsive **Color Utility Web App** built using **pure HTML, CSS, and Vanilla JavaScript**.

This project provides essential tools for designers, developers, and creators to work efficiently with colors — including a **Color Picker**, **Color Converter**, and **Beautiful Palettes Collection** in one clean interface.

> Built with simplicity. Designed for usability. Crafted by **ByteyBuddy**.

📌 Features

🎯 Color Picker
- Pick colors visually
- Get HEX, RGB, and HSL values instantly
- Copy color codes with one click

🔄 Color Converter
- Convert between:
  - HEX ⇄ RGB
  - RGB ⇄ HSL
- Real-time conversion
- Error handling for invalid inputs

🎨 Color Palettes
- Ready-made modern color palettes
- Great for UI/UX, websites, posters, and branding
- Copy colors directly from palettes

💎 Clean UI
- Minimal and modern design
- Fully responsive
- Smooth interactions
- Beginner-friendly code structure

---

🛠️ Built With

- **CSS3**
